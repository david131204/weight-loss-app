Project title: LeanPath

Short description:
LeanPath is a React Native and Expo mobile weight-loss app that allows users to calculate calorie targets, log food and activity, update weight and track progress.

Technologies:
React Native
Expo
Expo Router
Firebase Authentication
AsyncStorage

How to run:

Clone the repository.
Open the folder in VS Code.
Run npm install.
Run npx expo start.
Open the app using Expo Go on Android or iOS.

Notes:
The app uses Firebase Authentication for account access and AsyncStorage for user-specific local progress data.
